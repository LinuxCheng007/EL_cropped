华矩EL 透视裁剪工具  v4-UI + 检测内核 v3.8
=========================================
Win7 SP1 / Win10 / Win11  |  GPU 自动加速

使用步骤（右键 → 以管理员身份运行）：
  1. 1_download_python.bat    - 下载 Python 3.8（约 8MB，仅首次）
  2. 2_install_libraries.bat  - 安装依赖库（约 80MB，仅首次）
  3. 3_start_tool.bat         - 启动工具（每次使用）

检测内核 v3.8 改进：
  - 修复：正面拍摄时裁图顶边缺失（原版截掉顶部 8%，修复后 0%）
  - 修复：宽高比评分范围扩展，兼容极端透视仰角拍摄
  - 三路检测：亮度 Otsu（主）+ 纹理方差（备）+ 边缘检测（备）

外扩黑边滑块：
  0%  → 精确到检测边界（最紧）
  3%  → 默认，保留少量边框（推荐）
  10% → 保留较多边框

失败图片处理：
  - 失败图片出现在红色队列面板中
  - 点击任意失败图片打开手动编辑器
  - 拖动 4 个橙色圆点到面板准确角点
  - 点击「应用裁剪」完成处理

迁移到其他电脑：
  复制整个文件夹（包括 runtime\）
  在新电脑只需运行步骤 3

性能参考：
  GPU (NVIDIA/AMD/Intel): 300 张约 30~90 秒
  CPU 模式:               300 张约 3~5 分钟
